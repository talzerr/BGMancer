## Steam

This application uses the **Steam Web API** to fetch a user's public game library. No Steam login or OAuth is required from the user — only a server-side API key is used.

BGMancer does not access any private Steam data. Game library information is only accessible if the user has set their Steam profile's "Game details" visibility to **Public** in their Steam privacy settings.

Game cover images are loaded directly from Steam's public CDN (`cdn.akamai.steamstatic.com`) and are not stored by BGMancer.

By using the Steam import feature, you agree to the [Steam Web API Terms of Use](https://store.steampowered.com/stats/).

## Google OAuth / User Data

If you choose to sign in with Google, BGMancer requests permission to manage your YouTube playlists. This access is used solely to create and update the "BGMancer Journey" playlist on your account.

BGMancer does not sell, share, or store your Google account data beyond what is needed to authenticate your session. OAuth tokens are stored in memory only and are not persisted to the database.
