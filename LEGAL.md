## Steam

This application uses the **Steam Web API** to fetch a user's public game library. No Steam login or OAuth is required from the user — only a server-side API key is used.

BGMancer does not access any private Steam data. Game library information is only accessible if the user has set their Steam profile's "Game details" visibility to **Public** in their Steam privacy settings.

Game cover images are loaded directly from Steam's public CDN (`cdn.akamai.steamstatic.com`) and are not stored by BGMancer.

By using the Steam import feature, you agree to the [Steam Web API Terms of Use](https://store.steampowered.com/stats/).

## Google OAuth / User Data

If you choose to sign in with Google, BGMancer requests your email address for authentication. No profile name or picture is requested or stored.

If you use the "Sync to YouTube" feature, BGMancer requests additional permission to manage your YouTube playlists. This access is used solely to create and update the "BGMancer Journey" playlist on your account. YouTube permissions are only requested when you click Sync — not at initial sign-in.

BGMancer does not sell, share, or store your Google account data beyond what is needed to authenticate your session. OAuth tokens are stored in encrypted session cookies and are not persisted to the database.

## Fair use

BGMancer is a transformative, non-commercial curation tool. It does not reproduce, download, store, or distribute any copyrighted audio. It organizes and presents links to publicly available content already uploaded to YouTube.

This use may qualify as **Fair Use** under **17 U.S.C. § 107** on the following grounds:

1. **Purpose and character** — transformative curation for personal, non-commercial listening; no commercial gain.
2. **Nature of the work** — the underlying works are artistic, but the use is organizational, not reproductive.
3. **Amount used** — no audio files are copied or hosted. BGMancer indexes video IDs and surfaces them through YouTube's own embedded player.
4. **Market effect** — BGMancer drives traffic _to_ YouTube and to the rights holders' official content; it does not substitute for purchasing soundtracks.
